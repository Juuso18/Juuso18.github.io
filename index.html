Html on varmaan ensimmäinen kieli jonka opimme ja se on mielestäni kaikista helpoin näistä kaikista! Html on oleinen osa nettisivun teossa. Htmllällä voi tehdä kaiken mitä nettisivulle haluaa tietenkin javascriptin ja mahdollisesti css:än avustuksella mutta html on kuin pohja.

Css Seuraava koodi kieli jonka opimme oli css ja se on kanssa helppoa sanoisin että toisiksi helpoin näistä kaikista koodikielistä. Tällä nettisivulla voi muutaa kaikkea sivun olkonäkyyn liittyvää.

Seuraava koodikieli on javascript javascriptin opimme kolmantena ja tämä alkaa menemään jo vaikeaksi. Javascriptillä voi tehdä functioita ja kaikkea "automaattista" esimerkiksi laskin on tehtävissä javascriptillä.

Neljänneksi vaikein kieli on mielestäni Python ja sitä olemme oppineet vain vähän en tiedä siitä niin paljoa mutta haluaisn oppia lisää!
